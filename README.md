# youtube-downloader

YouTube Downloader (Tkinter GUI)

A simple desktop application with a graphical user interface (GUI) to download YouTube videos in different resolutions or extract audio only, built with Python and Tkinter.

Features
Download YouTube videos(quick and easy)
Choose video quality:
From 144p up to 1080p
Download audio only 
Live progress bar
Select custom save location
Clean and user-friendly interface
Uses threading to prevent UI freezing

Add a screenshot of your app here after uploading the project

Requirements

Make sure you have the following installed:

Python 3.8 or newer
Required library:
pip install pytubefix

Note: Tkinter usually comes pre-installed with Python


How to Use
Paste a YouTube video URL
Choose a save location
Select the desired quality
Click START DOWNLOAD
Wait for the download to complete 
Notes
Some qualities (like 1080p) may not include audio due to YouTube limitations
If a selected quality is unavailable, the app will prompt you to choose another
Make sure the URL is valid
